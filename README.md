# Shopper Spectrum Project

## Overview

Shopper Spectrum is a Machine Learning project that performs customer segmentation and product recommendation using the Online Retail dataset.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (15 Visualizations)
* RFM Analysis
* Customer Segmentation using KMeans Clustering
* Product Recommendation System using Cosine Similarity
* Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Streamlit

## Customer Segments

* VIP Customers
* Loyal Customers
* Regular Customers
* At Risk Customers

## Project Structure

* Notebook for analysis and model building
* Streamlit app for visualization
* Saved segmentation and recommendation outputs
